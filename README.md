# Fellfresse-Zattoo-EPG

## About
Provides a 14 Days XMLTV EPG for Zattoo DACH TV service.

## Usage
Grab the XMLTV file from this link and paste it into your favorite IPTV client:
```
https://raw.githubusercontent.com/Fellfresse/Zattoo-EPG/master/zattoo_de_epg.xml.gz
https://raw.githubusercontent.com/Fellfresse/Zattoo-EPG/master/zattoo_at_epg.xml.gz
https://raw.githubusercontent.com/Fellfresse/Zattoo-EPG/master/zattoo_ch_epg.xml.gz
```
